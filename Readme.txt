1. To view Test Cases open Test Cases > Test conditions.xlsx

2. To view full test report Open Report > Index.html

3. To view test execution recording open Recording > Automate todomvc.mp4

Steps for Execution:

1.Install Microsoft Visual Studio Community latest version
 
2.Download todomvc Solution from GitHub

3.Open the todomvc solution in Visual studio

4.Right click on the project and go to Manage NuGet Packages and install below packages.

Install Nuget pacakages:
Selenium.WebDriver (Version 4.5.1)
Selenium.Support (Version 4.5.1)
Selenium.WebDriver.ChromeDriver (Version 106) 
Selenium.Firefox.WebDriver (Version 0.27 )
DotNetSeleniumExtras.PageObjects (Version 3.11.0)
DotNetSeleniumExtras.WaitHelpers (Version 3.11.0)
ExcelDataReader (Version 3.6.0)
ExcelDataReader.DataSet (Version 3.6.0)
ExtentReports (Version 4.1.0)
Microsoft.NET.Test.Sdk (Version 17.3.2)
Microsoft.NETCore.App (Version 2.2.8)
NUnit (Version 3.13.3)
NUnit3TestAdapter (Version 4.2.1)
Selenium.WebDriver.GeckoDriver (Version 0.32.0)

5. In Visual studio open folder Config > Resource.resx and update the ExcelPath,ReportPath, ScreenShotPath to save files to your local pc

5. In Visual Studio open View > Test Explorer and run
